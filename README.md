# StockTracker
CLI-based Inventory Management System built with C# .NET Console App.

## Run
1. Open `InventoryManagementSystem.csproj` in Visual Studio
2. Press `F5`

## Features
- Categories and Suppliers management
- Add, View, Search, Edit, Remove Products
- Add and Deduct Stock
- Low Stock Alerts
- Inventory Value Report
- Transaction History
